# 신나는 리라이팅

**전부 새로 작성 예정**


## 계획
- 로깅:
    - 기본 로깅은 파이썬 내장 logging 사용
    - 로그 파일 날짜별 분리, colorlog 또는 rich로 하이라이팅
    - 에러 로깅은 sentry 사용 
- 백업:
    - tar,gz 압축 사용
    - tempfile 활용
- 기타:
    - koreanbots, top.gg용 sdk 직접 작성
    - dotmap 또는 nested dataclass 사용
- 개발환경:
    - python 3.9.5
    - poetry
    - docker
